# Portfolio Webpage

## How to view
- Open `index.html` in your browser.

## Customize
- Edit your name/links in `index.html` (search for `Your Name`, `you@example.com`, and `@yourhandle`).
- Update colors and spacing in `styles.css`.
- The contact form uses a `mailto:` link (no backend). Change the destination email in `script.js`.
